## Description  
"My-zootopia-api" is a simple Python script that fetches animal data from an external API and generates an HTML file listing animals with basic details (e.g. diet, location, type). The user types the name of an animal ‚Äî if data is found, the script produces `animals.html`, which can be opened in a browser to view the results.

## Requirements / Dependencies

- Python 3.x  
- API Key from the service ‚Äî you must get your API key from **api.ninjas.com**.  
- Python packages:  
  - `requests` (for HTTP requests)  
  - `json` (standard Python library)
